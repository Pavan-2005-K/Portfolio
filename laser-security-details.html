Semester 2 Electronics Project (Arduino)

Project Overview

We developed an Arduino-based security system designed to provide reliable, remote surveillance, primarily to address lapses during night shifts. The system utilizes a laser pointed at an LDR module. When the beam is broken, the circuit triggers immediate, multi-channel alerts.

Key Features and Technology

  • Control & Alerts: The Bluetooth module allows system control via a mobile app (Serial Bluetooth Terminal) and sends instant intrusion notifications to the phone.
  • Monitoring: An LCD display with an I2C module shows real-time status, including connection status and breach alerts ("Laser Broken!").
  • Intrusion Detection: A voltage change in the LDR module, caused by the interrupted laser beam, is detected by the Arduino, activating the buzzer and LED alerts.
  • Power: A 9V battery powers the system, with a buck converter stepping down the voltage to a safe 5V for all components.
